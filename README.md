# Customer Churn & Revenue Analysis Dashboard

## Project Overview
This project focuses on analyzing customer churn and revenue trends using Power BI. The dashboard helps identify customer retention patterns, revenue performance, customer behavior, and business growth insights through interactive visualizations and KPI tracking.

The objective of this project is to support data-driven decision-making by understanding why customers leave, which customer segments generate the highest revenue, and how business performance can be improved.

---

## Key Features
- Customer churn analysis
- Revenue trend monitoring
- Customer segmentation insights
- Interactive dashboard filters and slicers
- KPI cards for business metrics
- Dynamic charts and visual reports
- Data-driven business insights

---

## Technologies Used
- Power BI
- Power Query
- DAX
- Excel / CSV
- Data Visualization
- Business Intelligence

---

## Dashboard Insights
The dashboard provides insights into:
- Customer retention and churn patterns
- Revenue contribution by customer segments
- Monthly and yearly revenue trends
- High-risk churn customers
- Business performance indicators
- Customer behavior analysis
